Medium Clone - HTML & CSS 

Descrizione:
Questo progetto è una replica della pagina principale di Medium. 
È stato sviluppato utilizzando HTML e CSS, con l'uso di Flexbox per la gestione del layout e delle posizioni degli elementi. 
La pagina include sezioni come il menu di navigazione, un'area principale con articoli, una barra laterale e un footer.

Obiettivo: Riprodurre il design di una pagina di esempio di Medium utilizzando le tecniche apprese durante il corso di HTML, CSS, positioning e Flexbox.

Funzionalità:
-Header con logo e navbar.
-Sezione principale con il messaggio "Stay curious" e una sezione di articoli in evidenza.
-Cards dinamiche per gli articoli più letti.
-Footer con links per altre pagine.

Tecnologie utilizzate:
-HTML5: Per la struttura del contenuto.
-CSS3: Per lo stile e la disposizione.
-Flexbox: Per la gestione dei layout responsivi.
-Positioning: Per il posizionamento degli elementi.
